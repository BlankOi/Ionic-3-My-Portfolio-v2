As ususal, everything that you see is coming from API. To use Github API is a simple and easy process. You dont have to obtain the private key and what so ever, you just need a username.

First thing we need to do is create a provider to handle the request to the api, ionic provides a generator for this 

ionic g provider github

The thing that happend in this process is:

1. Im using a loader to act as an indicator that telling you that the user information and public repo is being fetch from github api.

2. After it returned the respond, I'm binding the information in a user and repo array so that I can use it in view.

3. If you tap on my repo, it will open up inappbrowser showing my repository
